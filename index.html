스마트폰 성능 검사

(주)리플러스의 폰집사는 스마트폰의 성능과 사양을 검사하는 강력한 검사도구입니다. 스마트폰을 사고 팔때 폰집사로 스마트폰의 성능을 검사하고 성능검사서를 공유하세요

안전한 데이터 삭제

폰집사 데이터 삭제 툴은 저장매체의 전 영역에 걸친 덮어쓰기 처리 기술을 이용해 안전하고 완벽하게 삭제합니다. 수리서비스를 선택하여 정밀 검수와 데이터 완전 삭제 서비스를 이용하세요

직거래 판매 지원

폰집사 앱을 사용하여 스마트폰의 상태를 검사한 후에는 손쉽게 직거래 할 수 있도록 도와드립니다. 성능검사서를 공유하거나, 고급 서비스 선택하여 수리와 세척을 거쳐 판매하세요.

정보유출 위협

- SSD의 TRIM 프로세스 : SSD에 블록을 지울 때 특정 데이터의 다시 쓰기를 건너뛸 수 있도록 정보를 주는데, 모든 블록은 다시 쓸 수 있는 횟수가 유한하기 때문에 SSD의 수명단축을 방지하는 중요한 프로세스입니다.
- 데이터삭제의 어려움 : SSD의 이러한 특성 때문에, 드라이브의 마모를 추가로 방지하기 위해 드라이브의 각 블록에 동일한 양의 읽기/쓰기 프로세스가 진행되도록 하기 위한 알고리즘이 있습니다. 이 프로세스는 마모 균등화로 불리며 드라이브가 작동 중일 때 자동으로 실행되는데, 문제는 데이터 삭제 시에도 마모 균등화로 인하여 기존의 데이터를 삭제하는 명령을 내려도 완전히 지워지지 않게 됩니다.
